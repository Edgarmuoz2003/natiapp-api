Este es el backend para el proyecto natiApp natillera digital

al descargargar el proyecto sigo los siguientes pasos:

1. Añadir las respectivas varibles de entorno en el .env
2. instalar postgre en el pc desde la pagina oficial
3. correr npm install
4. correr "npm run dev" para desarrollo o "npm start" para producción

NOTA:
produccion es la rama "master", para incorporar cambios usamos la rama latest que es la version mas reciente
